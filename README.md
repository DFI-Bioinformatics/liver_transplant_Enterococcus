# Enterococcus faecium Driven Metabolic Dysbiosis is Linked to Mortality following Liver Transplant


## Folder structure

1. **code**: contains all R scripts for generating boxplots, volcano plots and/or taxonomy barplots, etc. Scripts are organized by figure number. Resultant plots are saved in figures folder. 
2. **figures**: contains all unmodified graphs in PDF format. Graphs published along with the paper have been slightly rearranged and beautified in Adobe Illustrator.
3. **data**: contains raw or derived data for analysis and plotting.
